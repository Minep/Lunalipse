What is Parallax Design
Parallax Design is a new design that has been popularized since 2013.
It comes from the visual effect in 2D graphics especially in video game. And it also exists in our life, we can observe this phenomenon while we are moving. The phenomenon can be described as following: landscape which close to us move faster than the landscape which is far away from us.
It give a depth and 3D-impact to the web layout by justifying the velocity of movment of different layers when scrolling.

Notice of using NParallax
You may need to give a alternative ID to each element which you hope them to give parallax image presentation.
Initialize NParallax
/* Put them into page onload event */ $("body").NParallax({ imgParallax:["#i1","#i2","#i3"], velocity:0.2, speed:0.6 })
Keyword Explanation
imgParallax Array of ID of parallax images container
velocity Velocity of parallax when scrolling (MUST LESS THAN 1 )
speed speed of parallax offset per velocity
